# Welcome to Voult.dev

> Developer-first authentication infrastructure for modern applications.

Voult.dev is an authentication platform built to simplify secure user management for developers and startups. Voult provides scalable APIs, authentication flows, and security tooling so teams can ship products faster without rebuilding auth from scratch.

---

## What Voult Provides

- Secure authentication APIs
- Email/password authentication
- OAuth providers
- Magic links & passwordless authentication
- JWT & session management
- Email verification & password resets
- Developer-friendly SDKs
- Scalable backend infrastructure
- Modern developer experience

---

## Why We’re Building Voult

Authentication is one of the most critical parts of any application, but building it correctly is difficult, time-consuming, and security-sensitive.

Voult exists to give developers:

- Faster product development
- Production-ready authentication flows
- Better security defaults
- Clean APIs and SDKs
- Infrastructure that scales with growth

Our goal is to make authentication infrastructure simple, reliable, and developer-centric.
